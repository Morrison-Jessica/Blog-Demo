## Highlights / Skills Demonstrated

- Static site generation with **Jekyll**  
- Writing and structuring posts in **Markdown**  
- Custom layouts (`default`, `post`) and reusable components  
- **TailwindCSS** for responsive, modern styling  
- Proper internal linking using **relative_url**  
- Deployment and hosting on **GitHub Pages**  


## Features

- **Static site** with posts written in Markdown
- **Responsive design** using TailwindCSS
- **Custom layouts** (`default` and `post`) for consistent styling
- **Automatic listing of posts** on the home page
- **Navigation** between Home, About, and Posts pages
- **GitHub Pages deployment** with proper relative URLs


## Technologies

- **Jekyll** for static site generation
- **Markdown** for post content
- **TailwindCSS** for styling
- **GitHub Pages** for hosting


## Setup / Deployment

```bash````
# Clone repo
